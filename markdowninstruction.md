# Инструкция по MD

## Выделение текста

Чтобы выделить текст, его нужно обрамить с двух сторон звездочками (*).
Для курсива используется по одной. Например, *вот так*.

Для полужирного шрифта используются две звездочки (**), и выглядят они **вот так**.
